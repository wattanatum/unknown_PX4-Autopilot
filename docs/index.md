# Unknown PX4 Autopilot

Unknown PX4 Autopilot is a ROS 2 Jazzy and PX4 SITL simulation project for autonomous drone mapping and navigation using Gazebo Harmonic, SLAM Toolbox, Nav2, and PX4 offboard control.

## Project Overview

This project integrates PX4 SITL with ROS 2 Jazzy to simulate an autonomous drone in Gazebo Harmonic. The drone can perform SLAM mapping, autonomous navigation, and velocity control through ROS 2 Nav2 and PX4 offboard control.

## Features

- PX4 SITL drone simulation
- Gazebo Harmonic environment
- ROS 2 Jazzy integration
- SLAM Toolbox mapping
- Nav2 autonomous navigation
- PX4 offboard control
- Laser scan and odometry bridge
- RViz2 visualization
- Custom Gazebo world for outdoor mapping and navigation

## Main Technologies

- ROS 2 Jazzy
- PX4 Autopilot
- Gazebo Harmonic
- Nav2
- SLAM Toolbox
- MAVSDK
- RViz2
- Python
